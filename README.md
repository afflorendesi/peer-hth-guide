# peer-hth-guide

## 项目简介

`peer-hth-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，而是作为通用 HTML 页面存储与分发空间，便于在不同场景下引用、预览或分享。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放各独立 HTML 页面，文件名建议使用英文小写、连字符分隔，便于 URL 引用。
- 仓库根目录下的 `index.html` 可作为入口或导航页面，非必需。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文件（不含外部依赖或仅依赖公共 CDN 资源）。
- 页面内容以信息展示、工具界面、说明文档等静态 HTML 为主，不含后端逻辑。
- 归档目的：便于长期保存、离线查看、快速分享及版本回溯。

## 使用方式

1. **直接浏览**：通过 GitHub Pages 或本地打开 HTML 文件即可查看。
2. **引用链接**：将页面 URL 直接分享或嵌入到其他文档中。
3. **本地预览**：`git clone` 后，双击 HTML 文件或在浏览器中打开。

## 维护说明

- 欢迎提交新的独立 HTML 页面至 `pages/` 目录，请确保文件命名清晰、内容完整。
- 建议在提交时附带简要说明（可在 commit message 或 PR 描述中注明页面用途）。
- 本仓库不收录涉及敏感、违法或恶意内容的页面。
- 如需修改或删除已有页面，请提交 PR 或 Issue 说明理由。

## 贡献

- Fork 本仓库 → 添加或修改页面 → 提交 Pull Request。
- 如有疑问或建议，欢迎开启 Issue 讨论。
